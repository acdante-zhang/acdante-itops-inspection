# DESIGN.md

## 项目与用户画像
- DBA 和运维人员使用的 IT 运维巡检平台
- 需要快速巡检多类型设备、定位问题，操作要高效、信息要精准
- 支持企业数据中心、混合云环境

## Design Tokens

### 色彩
- 主色调：深色背景（slate-950/900），搭配青色(cyan)作为品牌色，琥珀色(amber)作为告警色
- 状态色标：
  - OK/正常 = 绿色 (green-400/500)
  - WARNING/告警 = 琥珀色 (amber-400/500)
  - CRITICAL/严重 = 红色 (red-400/500)
  - RUNNING/执行中 = 青色 (cyan-400/500)
  - PENDING/待执行 = 蓝色 (blue-400/500)
- 设备类型色标（标签）：
  - Linux = 系统默认
  - Oracle = 琥珀色
  - MySQL = 蓝色
  - PostgreSQL = 靛蓝色
  - Network = 青色
  - Storage = 绿色
  - BMC = 灰色
  - SAN Switch = 紫色

### 字体
- 系统字体栈（font-sans），等宽字体用于命令/SQL显示（font-mono）

## 布局与响应式
- 左侧固定侧边栏（240px）+ 右侧主内容区
- 侧边栏可折叠分组
- 表格数据密度高，行高紧凑
- 报告详情采用左右分栏（列表+详情）

## 交互规范
- 只读命令显示绿色锁图标，非只读命令显示琥珀色警告图标
- 连接测试使用旋转动画反馈
- 任务执行状态使用实时进度条
- 删除操作需二次确认
